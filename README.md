# 🍔 Food Delivery System

Simple Java project using OOP concepts.  
Implements Pizza, Sandwich, and Salad orders with different pricing rules.  
Calculates total cost and delivery time based on user input.

## Features
- Abstraction and Inheritance with `FoodOrder` class  
- Pizza → Rs.200 per extra topping  
- Sandwich → Rs.150 per extra filling  
- Salad → 10% discount for large portion  
- Calculates total price and estimated delivery time  

## How to Run
1. Open each `.java` file in a Java IDE  
2. Compile all files  
3. Run `FoodDeliverySystem.java`  
4. Follow the prompts to place your order
